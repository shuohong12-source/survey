# 问卷调查系统

这是一个 Python 课程实践项目，使用 Flask + SQLite 实现在线问卷调查系统。

## 功能

- 用户注册、登录、退出
- 创建问卷：标题、说明、题目列表
- 支持单选题、多选题、填空题
- 发布问卷后生成填写链接，安装 `qrcode[pil]` 后自动生成二维码
- 在线填写问卷并提交答案
- 查看统计结果：选项票数、填空答案、回收率、平均答题时间
- 问卷列表：我的问卷、公开问卷
- SQLite 数据存储
- 扩展功能：逻辑跳转、matplotlib 图表、模板复用、CSV/Excel 导出、开放/截止时间、匿名投票

## 运行方式

网页版：

```bash
pip install -r requirements.txt
python app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

首次运行会自动创建 `survey.db` 数据库。

桌面 GUI 版：

```bash
python gui.py
```

GUI 版使用 Tkinter 编写，和网页版共用同一个 `survey.db` 数据库。

## 逻辑跳转

创建问卷时，每个选项都有“跳转题号”输入框。需要逻辑跳转时，在对应选项后填写目标题号即可。

```text
满意    跳转题号：4
一般
不满意  跳转题号：3
```

表示选择“满意”后跳到第 4 题，选择“不满意”后跳到第 3 题。
